Lesson 2: Comments And Attributes

Comments

When you put code in a comment, it disables the code. This is useful for leaving notes to yourself or collaborators, or for keeping code you aren't sure about using in a way that won't affect the rest of your webpage. <!-- This is a correctly-written comment in HTML.-->. If you were to copy-paste the previous sentence into a text-editor, it would not show up on a webpage because it is enclosed in a comment.

Attributes

Tags often have attributes that modify the tag. Notice how this paragraph has dark blue text. This can be added with a style attribute inside a paragraph attribute. That would look like this: <p style="color:darkblue"> This will have dark blue text </p>.
