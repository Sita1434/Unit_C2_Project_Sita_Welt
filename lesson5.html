Lesson 5: Tables

Required Tags

Making a table requires three tags. The tags and their purposes are outlined below:
  1. <table>: creates a table.
  2. <th>: creates a table row formatted as a heading.
  3. <tr>: creates a table row.
  4. <td>: adds the data intended for a table cell.

You may also need the attributes "colspan" (to have a cell span multiple columns) and "rowspan" (to have a cell span multiple rows). Below is a table designed in HTML:

Period Class (A Day) Class (B Day) Teacher (A Day) Teacher (B Day)
1 Spanish Sra. Africano
2 Global History II Mr. Caccamo
3 Computer Science Health Mr. Saha Dr. Helfenbein
4 PE Mr. Powell
5 English Ms. Gallo
6 Lunch N/A
7 Algebra II Ms. Mun
8 Physics Mr. Laskowski
Here is all the code needed to create that table: 
      <table>
        <tr>
          <th>Period</th>
          <th>Class (A Day)</th>
          <th>Class (B Day)</th>
          <th>Teacher (A Day)</th>
          <th>Teacher (B Day)</th>
          </tr>
        <tr>
        <td>1</td>
        <td colspan="2">Spanish</td>
          <td colspan="2">Sra. Africano</td>
        </tr>
        <tr>
        <td>2</td>
          <td colspan="2">Global History II</td>
          <td colspan="2">Mr. Caccamo</td>
        </tr>
        <tr>
        <td>3</td>
          <td rowspan="2">Computer Science</td>
          <td>Health>/td>
          <td rowspan="2">Mr. Saha</td>
            <td>Dr. Helfenbein</td>
            </tr>
            <tr>
            <td>4</td>
            <td>PE</td>
            <td>Mr. Powell</td>      
            </tr>
            <tr>
            <td>5</td>
          <td colspan="2">English</td>
          <td colspan="2">Ms. Gallo</td>
            </tr>
            <tr>
            <td>6</td>
          <td colspan="2">Lunch</td>
          <td colspan="2">N/A</td>
            </tr>
            <tr>
            <td>7</td>
        <td colspan="2">Algebra II</td>
        <td colspan="2">Ms. Mun</td>
          </tr>
          <tr>
          <td>8</td>
        <td colspan="2">Physics</td>
        <td colspan="2">Mr. Laskowski</td>
          </tr>
      </table>

To make the borders appear, you will also need to add the following CSS code:

table, th, td {
      Border:1px solid black;
    }
