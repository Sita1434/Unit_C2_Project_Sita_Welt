Lesson 1: Creating a Basic HTML Webpage

As you have already learned, HTML is a markup language. Markups in HTML are called tags. Tags begin and end with angle brackets, and any commands/codes you want to write in HTML must be written in the form of a tag. Below, you will learn some of the most important and essential HTML tags, the ones you can't make a webpage without.

Essential HTML Elements

Every single HTML page you create, no matter what, has to have certain elements in order to work. This section will go over these elements.

First of all, each HTML page must start with <!DOCTYPE html>. This is called an "HTML5 Doctype Declaration", and it essentially tells your text editor what language you are using (in this case, HTML Version 5).

Directly under that tag, you must put the <html> tag, which begins the page. Within the HTML tag, you include the <head> and <body> tags. The <head> tag includes all the page's metadata. We'll get more into what exactly that means in a later lesson. For now, all you need to know is the head holds everything that the webpage and web crawlers need to know, but that should not be visible on the actual webpage.The <body> tag goes after the head tag and includes everything that is visible on the webpage. Every image, word, heading, link, etc., that shows up on the webpage is written in the body tag of the webpage.

Within the body tag, there are a variety of other tags you can and should use in your webpage. To add headings, you can use a heading tag. The largest heading is <h1>, the next largest is <h2>, and so on and so on until the smallest heading, <h6>.

Within each heading. or subheading, you may very well want to add paragraphs. To do so, use the <p> tag. Heading and paragraph tags are known as block-level elements, because using them creates a new block of text (like a paragraph). There are also inline elements, which can appear within a paragraph, inside and/or next to other elements.

Text Formatting

Most text formatting tags are inline elements. With HTML, you can make words bold with the <strong> or <b> tags, italicized with the <i> or <em> tags, highlighted with the <mark> tag, smaller with the <small> tag, crossed out with the <del> tag, underlined with the <ins> tag, subscript with the <sub> tag, or superscript with the <sup>.

Lists

Making lists in HTML is relatively simple. You can make either a numbered list or a bulleted one. Here is what the code for making a numbered list looks like:

<ol>
     <li>Item 1</li>
     <li>Item 2</li>
     <li>Item 3</li>
</ol>
Wrting this code will show this on your webpage:
  1. Item 1
  2. Item 2
  3. Item 3

To make an unordered list with bullets, replace "ol" with "ul".
