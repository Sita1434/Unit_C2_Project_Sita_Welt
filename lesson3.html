Lesson 3: Adding Links, Images, and Iframes

Links

Adding HTML links can be done using the <a> tag. Adding a link requires a "href" attribute to be added to the anchor (<a>) tag. The general syntax for adding an HTML link is: <a href="pathToFile">LinkText</a>. For an external link (linking something from the web, for example) the "pathToFile" will be the URL of whatever you want to link. For an internal link (linking something from your computer), put the relative file path of the linked file in the "pathToFile". To link to google.com, your link code would look like this: <a href="https://google.com">google.com</a> and would appear on the webpage as google.com.

Images

Images can be linked in HTML using the <img> tag with the "src" attribute. See the photo shown below of a map of the US. This photo was added using this code: <img src="actualUSA.jpeg"width="100"alt=USAmap>. The "width" tag specifies the width of the image in pixels. Using a "height" attribute instead will define the height. Unless both width and height are specified, proportions will remain constant. The "alt" attribute holds the description of the image to be displayed if the image cannot show. It also provides useful informatin for screenreading technology.
